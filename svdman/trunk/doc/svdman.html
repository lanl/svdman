Aims:
    o Perform svd on a matrix of gene expression data
    o Identify gene groupings for each of the modes of the svd
    o Check for self-consistency of the svd model
    o Output results in a format easily imported into other software applications.
    o Design for use as a modular part of a larger system.
Requirements:
    o Tab-delimited ASCII data file, with labels across the top and at the left. The first element of the file is blank (i.e. the first character is a tab).
Installation:
      Download the distribution.
      Unpack the distribution via: gzip -dc svdman.tgz | tar xf -
      This will create a directory svd/. Change directory to svd/src/
      Build the binaries by typing "make".
      Test svdman using data provided in the tst/ directory. Use the shell script svdman_test.csh.
      See svdman.cc documentation for a description of how to use svdman.
